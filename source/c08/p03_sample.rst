**函数计算**

1. 求三个数之和

.. code-block:: python

   def sum(a, b, c):
       return a + b + c
   
   
   result = sum(1, 2, 3)
   print(result)  # 6




2. 求三个数平均值

.. code-block:: python

   def average(a, b, c):
       sumResult = sum(a, b, c)
       return sumResult / 3
   
   result = average(1, 2, 3)
   print(result)  # 2.0
